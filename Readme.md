# FloraVision вЂ“ Frontend Developer Internship Assignment

## Overview

FloraVision is a pixel-perfect frontend implementation of a modern plant e-commerce landing page designed from the provided Figma design.

This project was developed as part of the Frontend Developer Internship Test using React.js, Tailwind CSS, and CSS3.

The implementation focuses on:

* Pixel-perfect UI matching
* Responsive design
* Reusable React components
* Glassmorphism effects
* Clean layout structure
* Modern frontend practices
* Accessibility and responsiveness

# Installation & Setup

## Clone Repository

```bash
git clone <your-github-repo-link>
```

---

## Navigate to Project

```bash
cd project-folder
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```
